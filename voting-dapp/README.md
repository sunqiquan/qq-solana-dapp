# voting-dapp

This is a solana voting program and the app folder is the next.js webpage application

## Getting Started

### Installation

####

```shell
anchor build
```

#### deploy

```shell
anchor deploy
```

#### test

```shell
anchor test --skip-local-validator --skip-deploy
```

#### copy idl to app/idl

```shell
cp target/idl/voting_dapp.json app/idl/
```
